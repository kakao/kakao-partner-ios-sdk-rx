# RxKakaoPartnerSDKLink

RxSwift를 사용하는 파트너용 카카오링크 모듈입니다.

## Requirements
- Xcode 11.0
- iOS 11.0
- Swift 5.0
- CocoaPods 1.8.0

## Dependencies
- RxKakaoSDKCommon
- RxKakaoSDKLink
- KakaoPartnerSDKLink

## Installation
```
pod 'RxKakaoPartnerSDKLink'
```

## Import
```
import RxKakaoPartnerSDKLink
```
