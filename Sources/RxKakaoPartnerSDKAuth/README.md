# RxKakaoPartnerSDKAuth

RxSwift를 사용하는 파트너용 사용자 인증 및 토큰 관리 모듈입니다.

## Requirements
- Xcode 11.0
- iOS 11.0
- Swift 5.0
- CocoaPods 1.8.0

## Dependencies
- RxKakaoSDKCommon
- RxKakaoSDKAuth
- KakaoPartnerSDKAuth

## Installation
```
pod 'RxKakaoPartnerSDKAuth'
```

## Import
```
import RxKakaoPartnerSDKAuth
```
